## Specifications

Use it to indicate that a part of the screen is reactive to slide touch events.

```jsx
import Grip from '@blablacar/ui-library/build/grip'
<Grip
  onSlideUp={() => {
    /* ... */
  }}
  onSlideDown={() => {
    /* ... */
  }}
>
  <TheVoice>What is your route?</TheVoice>
</Grip>
```
