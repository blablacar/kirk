## Specifications

Used to display multiple TripCards in a grid pattern. Behaves the same as `CardsSection` on small devices.

## Usage

```jsx
import { CardsGridSection } from '@blablacar/ui-library/build/layout/section/cardsGridSection'
;<CardsGridSection>
  <TripCard />
  <TripCard />
  <TripCard />
</CardsGridSection>
```
