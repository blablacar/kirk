## Specifications

Highlight Section applied mainly on Landing Pages
Follows a _3 columns Grid System_: stacked moving up to equal columns

#### HighlightContentItems

Heading + Items
SEO: Items are a list of elements `li`

#### HighlightItems

Override `ItemChoice` with highlight specifications: 2 lignes, overflow, centered alignement;

## Usage

```jsx
import { HighlightSection } from '@blablacar/ui-library/build/layout/section/highlightSection'

const items = [{}]
<HighlightSection title="Section title"  items={items}>Section content</HighlightSection>
```
