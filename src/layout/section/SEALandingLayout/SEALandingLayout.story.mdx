## Specifications

SEA Landing Pages Half Width Column Vertical Grid.

`Body` tracker has max-with `375px` and is vertical/horizontally centered
`Figure` tracker is always `HeroSection`

## Usage

```jsx
import { SEALandingLayout } from '@blablacar/ui-library/build/halfSection'

 <SEALandingLayout
    heroText="Et vous, qui allez-vous retrouver ?"
    heroImageUrlSmall={ILLUSTRATIONS['carpool-bla-bla-bus'].small}
    heroImageUrlLarge={ILLUSTRATIONS['carpool-bla-bla-bus'].large}
  >
    // Content here {children}
  </SEALandingLayout>
```
